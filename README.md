# SteadyTenant Portal

A simple maintenance request tool built for small landlords managing under 100 units.

Workflow: Tenants submit issues with photos, landlords track everything from a central dashboard. Simple and centralized low-level ticketing system.

# Business Case
This is a focused, simple alternative for smaller landlords, who don't want to invest or use big and complex property management platforms.

# Features (in progress)
- [x] Tenant request submission with photo upload
- [x] Priority levels: Low, Medium, Urgent
- [x] Landlord dashboard with real-time ticket tracking
- [x] Status updates: New → Scheduled → In Progress → Done
- [x] Photo thumbnails with lightbox viewer
- [x] Hide resolved tickets by default
- [x] Landlord authentication (login/logout)
- [ ] Email notifications
- [ ] Tenant status tracking page

# Built with
- HTML, CSS, JavaScript
- Supabase (database + storage + auth)
- GitHub Pages (deployment)
