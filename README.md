# SteadyTenant Portal

A simple maintenance request tool built for small landlords managing under 100 units.

Workflow: Tenants submit issues with photos, landlords track everything from a central dashboard. Simple and centralized low-level ticketing system.

# Business Case
This is a focused, simple alternative for smaller landlords, who don't want to invest or use big and complex property management platforms.

## Features (in progress)
- [ ] Tenant request submission with photos
- [ ] Landlord dashboard with status tracking
- [ ] Priority levels: Low, Medium, Urgent
- [ ] Contractor/assignee field
